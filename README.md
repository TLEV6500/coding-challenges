## 🚀 Getting Started

Follow these steps to explore or run the solutions locally.

### Prerequisites

Ensure you have [Bun](https://bun.com) installed on your local machine:
```bash
curl -fsSL https://bun.sh | bash
```

### Installation & Setup

1. Clone the repository:
   ```bash
   git clone https://github.com
   ```
2. Navigate into the directory:
   ```bash
   cd coding-challenges
   ```
3. Install dependencies:
   ```bash
   bun install
   ```

### Running Solutions

~~To execute a specific script or solution file, use the fast `bun run` command:~~
```~bash
bun run path/to/challenge.ts
```

### Running Tests

If you have test suites configured, execute them instantly using Bun's native test runner:
```bash
bun test
```
